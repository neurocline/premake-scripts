Premake for libgit2
===================

Clone libgit2

```
git clone git@github.com:libgit2/libgit2.git
cd libgit2
```

Copy the premake scripts from this folder to the libgit2 root.

Build.

```
premake5 vs2013
```

Analyzing libgit2
-----------------

As of May 2015, libgit2 was built with CMake.
